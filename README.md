💬 Xtra Fine

Xtra Fine is a real-time chat application built to make conversations feel simple, fast, and human — without unnecessary clutter.

This project was designed, developed, and maintained independently by Aryan, under the personal development identity XF.

✨ Features

🔥 Real-time messaging using WebSockets (Socket.IO)

📱 Responsive UI — works smoothly on mobile and desktop

🧼 Clean, minimal interface focused on conversations

🔐 Authenticated sessions with protected routes

⚡ Optimistic UI updates for instant feedback

🧭 Thoughtful navigation and UX flow

🛠 Tech Stack
Frontend

React

React Router DOM

Redux Toolkit

Tailwind CSS

Framer Motion

Backend

Node.js

Express

MongoDB

Socket.IO

📁 Project Structure
├── backend/        # Express + Socket.IO server
├── frontend/       # React client
└── README.md


Both frontend and backend live in the same repository for easier development and deployment.

🚀 Getting Started (Local Setup)
1. Clone the repository
git clone https://github.com/Roaring-Moon0/xtra-fine-chat.git
cd xtra-fine

2. Install dependencies
# Backend
cd backend
npm install

# Frontend
cd ../frontend
npm install

3. Environment variables

Create .env files for both frontend and backend with the required values
(e.g. database URL, ports, etc.).

4. Run the app
# Backend
cd backend
npm run dev

# Frontend
cd ../frontend
npm run dev
